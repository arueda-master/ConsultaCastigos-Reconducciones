# Consulta Dinámica de Créditos: Castigos y Reconducciones

## 📝 Descripción
Esta herramienta automatiza la extracción y consulta de información financiera crítica desde la base de datos. Permite obtener de manera dinámica y eficiente el detalle de **créditos castigados** y **reconducciones**, aplicando filtros de temporalidad para optimizar el consumo de recursos.

## ✨ Características Principales
* **Conexión Directa:** Integración segura con la base de datos institucional.
* **Consultas Dinámicas:** Capacidad de filtrar por periodos específicos de tiempo.
* **Optimización de Datos:** Consumo selectivo de información (limita la carga de datos para mejorar el rendimiento).
* **Segmentación:** Diferenciación clara entre flujos de castigos y reconducciones.

## 🛠️ Tecnologías
* **Lenguaje:** [Ej: Python / Node.js]
* **Base de Datos:** [Ej: SQL Server / PostgreSQL / Oracle]
* **Librerías principales:** [Ej: Pandas, SQLAlchemy, pyodbc]
